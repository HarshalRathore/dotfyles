# Muktesh Mishra

**Muktesh Mishra** is a lead engineer for Applied AI for Developer Platforms at [[entities/adobe|Adobe]]. ^[extracted]

## Contributions

At AIEF2025, Muktesh presented on running evaluations at scale beyond accuracy and similarity metrics, covering adaptive eval strategies for different AI application types (RAG, code generation, agents), data-centric eval design, and the trade-off between human-in-the-loop fidelity and automated speed. ^[extracted]

He is also a co-author of the CI/CD Design Patterns book and has been involved in open-source work across the developer communities. ^[extracted]
