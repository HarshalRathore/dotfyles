# Zhu Liang

Zhu Liang is an AI researcher based at Ohio State University who sparked a viral Twitter discussion about using AI agents as evaluators instead of LLMs.

## Twitter Activity (Jul 2026)

On July 4, 2026, Zhu Liang tweeted the question that catalyzed the Mind2Web 2 discussion:

> "what if instead of llm as judge, we use agent as judge, for evals? anyone working on this?"

**Engagement:** 60 replies, 9 reposts, 245 likes, 75,000 views — demonstrating significant community interest in agent-based evaluation.

The next day, [[entities/yu-su|Yu Su]] replied by sharing the Mind2Web 2 paper, which directly proposed the "Agent-as-a-Judge" framework — exactly what Zhu Liang was asking about. Zhu Liang then replied "nice to see papers that were ahead of its time" (4 likes, 4.2K views).
