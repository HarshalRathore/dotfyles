# Minute (AI Transcription)

**Minute** is an AI transcription and summarization tool built by the **Incubator for AI** (i.AI) for secure government meeting documentation. ^[extracted]

## Use Cases

Minute targets government scenarios where secure AI transcription and summarization could be transformational — particularly for frontline staff who spend time on administration, paperwork, and forms instead of their core work. ^[extracted]

## Feature Exploration Approach

Minute exemplifies the "go wide with features" strategy. i.AI knew there were many ways to deliver a streamlined transcription experience, and many different AI features that could help. Rather than committing to one design, they:

1. Used **AI coding assistance** to build many different features rapidly
2. Tested those features with different groups of users
3. Stripped back to focus on what actually worked

The key insight: using AI coding tools to build features means you don't develop sentimental attachment to them, making it easier to cut features that don't resonate with users. ^[extracted]

## Feature Examples

Minute initially included multiple feature variations:

- Template selection for different output formats
- Agenda input for meetings with structured agendas
- AI edit button for free-text output editing
- AI chat for asking questions about the meeting

These were later streamlined based on what worked with real users. ^[extracted]
