# Microsoft’s eShop Reference Application

## Basic Information
- Type: project
- Source: microservices-demo-projects_ac9b1f

## Description
Microsoft’s eShop Reference Application is an open-source sample e-commerce platform built with .NET/C# and a microservices architecture. The project is a primary showcase for deploying .NET applications using modern containerization with [[concepts/bridge-network-in-docker]] and orchestration with Kubernetes. It functions as a comprehensive, multi-service demo application, featuring a web frontend and several backend services, packaged for cross-platform deployment. The repository provides ready-to-use configuration for Docker Compose, Kubernetes manifests, and Helm charts, making it a practical resource for developers learning enterprise-like deployment patterns. Formerly known as eShopOnContainers, it is now maintained under the GitHub repository `dotnet/eShop`.
