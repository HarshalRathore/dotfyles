# Sam Bhagwat

Co-founder of [[entities/mastra-ai|Mastra.ai]] and previously [[entities/gatsby|Gatsby]]. Author of the book *Principles of AI Agents*. Presented at AI Engineer World's Fair 2025 on the relationship between agents and workflows, arguing the field should compose both patterns rather than choose one. ^[extracted]

## Background

- Co-founded [[entities/gatsby|Gatsby]], a React meta-framework that used GraphQL as its default data-fetching layer ^[extracted]
- Co-founded [[entities/mastra-ai|Mastra.ai]], a workflow framework for AI agent development ^[extracted]
- Authored *Principles of AI Agents* ^[extracted]

## Key Positions

- **Agents and workflows are complementary** — the real value comes from composing agents as workflow steps, workflows as agent tools, supervisor agents, and nested workflows ^[extracted]
- **Fluent code readability over graph theory** — workflow frameworks should use readable sequential syntax, not nodes-and-edges graph APIs that require developers to learn graph theory ^[extracted]
- **Community of practice over theory** — AI engineering practice evolves faster than theory; if your agent works with 20 tools despite conventional wisdom, the practice is correct, not the theory ^[extracted]
- **Power vs control trade-off** — start with agent-driven flexibility (power), add structured workflows (control) where reliability demands it ^[extracted]
