# Vercel

Vercel is a cloud platform for frontend hosting and AI tooling. It creates developer tools including the Vercel AI SDK, v0 (v0.dev AI UI generation), and vZero (full-stack vibe coding platform). ^[extracted]

## Products

- **Vercel AI SDK** — SDK for building AI-powered applications with streaming text support. Used by [[entities/ido-pesok|Ido Pesok]]'s team for production LLM integrations. ^[extracted]
- **v0 (v0.dev)** — AI-powered UI generation tool. Mentioned alongside Cursor, Copilot, Windsurf, and Bolt as part of the wave of AI coding tools. ^[extracted]
- **vZero** — Full-stack vibe coding platform. Crossed 100 million messages sent at the time of AIEF2025. ^[extracted]
- **Fluid Compute** — Serverless compute platform for deploying AI applications. ^[extracted]
