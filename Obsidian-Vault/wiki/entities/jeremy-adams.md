# Jeremy Adams

Jeremy Adams is a member of the ecosystem team at [[entities/dagger|Dagger]]. He co-presented the hands-on workshop "Ship Agents that Ship" at the AI Engineer World's Fair 2025 (AIEF2025), alongside [[entities/kyle-penfound|Kyle Penfound]]. ^[extracted]

## Background

Adams has been at Dagger for several years and has observed the evolution of Dagger usage from classic CI/CD pipelines to AI agent workflows. He characterizes Dagger as providing "daggers for software engineering workflows and environments" — containerized environments with functions that can be used by both human developers and AI agents. ^[extracted]

## Key Contributions

At AIEF2025, Adams provided framing and context for the workshop:

- **"Robot body and brain"** metaphor — agent environments (tools, inputs, outputs, state) as the robot body, pluggable LLMs as the brain ^[extracted]
- **Containerized environments for agent isolation** — emphasizing that agent execution happens entirely in containers, not on the host filesystem, enabling safe parallel runs ^[extracted]
- **Role of platform engineering** — the same Dagger workflows that serve CI/CD can be handed to agents, with predetermined scope and deterministic validation ^[extracted]
- **Dagger as the middle ground** — providing guardrails (scoped tools, typed interfaces, test-driven validation) while letting agents run autonomously ^[extracted]
