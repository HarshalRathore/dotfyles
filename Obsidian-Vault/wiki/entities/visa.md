# Visa

Visa is a global payments technology company developing infrastructure for agentic commerce. ^[extracted]

## Delegated Authentication

Visa is working on delegated authentication, an elegant solution for AI agent checkout where the agent can use a user's actual credit card and go through the checkout flow on their behalf — rather than the current approach where a software provider spins up a virtual card to buy items on the user's behalf (the Stripe SDK model). ^[extracted] This represents a shift from low-level payment infrastructure to higher-level intent infrastructure in the agentic commerce stack. ^[inferred]
