# LengthMist

**LengthMist** is an observability tool for managing offline evaluation runs of AI systems. It tracks experiments over time, enabling teams to see how their system's performance changes at different time steps during evaluation. ^[extracted]

## Usage in Tavily's Pipeline

LengthMist is used by Tavily to track the experiments generated by their dynamic eval set agent. It provides the observability layer that connects the automated Q&A generation pipeline to performance tracking and regression detection. ^[extracted]
