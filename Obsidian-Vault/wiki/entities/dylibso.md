# Dylibso

**Dylibso** is a company that builds tools for the [[concepts/model-context-protocol|MCP]] ecosystem. They are creators of [[entities/mcp.run|mcp.run]], a platform for running MCP clients and servers at scale.

## mcp.run

[[entities/mcp.run|mcp.run]] is Dylibso's flagship product — a platform for hosting and managing [[concepts/model-context-protocol|MCP]] servers and clients. Key features:

- **Profiles** — compose multiple MCP servers into a single virtual server
- **Task client** — single-prompt agent triggerable via URL or schedule
- **OTEL export** — exporting telemetry to [[concepts/open-telemetry|OpenTelemetry]]-compatible sinks (announced at AIEF2025)

## Observable.tools

Benjamin Eckel of Dylibso co-created [[entities/observable.tools|observable.tools]], a manifesto advocating for vendor-neutral, standardized observability in the [[concepts/model-context-protocol|MCP]] ecosystem. ^[extracted]
