# Richmond Alake

Richmond Alake is a speaker and representative at [[entities/mongodb|MongoDB]], known for presenting on agent memory architecture and the role of memory in building reliable AI agents.

## AIEF2025 Talk

Richmond delivered the talk **"Architecting Agent Memory: Principles, Patterns, and Best Practices"** at AI Engineer World's Fair 2025, arguing that memory is the single most important component for building believable, capable, and reliable AI agents. The talk covered the agenticity spectrum, memory types, memory management systems, and MongoDB's positioning as the memory provider for agentic systems.

## Key Positions

- Memory is the foundation of agent intelligence, directly analogous to human intelligence
- The agenticity spectrum is more useful than binary agent/non-agent definitions
- Prompt engineering focus will diminish as persistence-based memory becomes dominant
- Retrieval-as-tool (agentic RAG) is the next evolution beyond system-level RAG
