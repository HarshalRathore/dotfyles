# W3Schools

W3Schools is a web development tutorial website that [[entities/greg-brockman|Greg Brockman]] used to learn PHP after high school. ^[extracted]

Brockman went to W3Schools to learn how to build a website for a chemistry textbook he had written. His first project was a table-sorting widget — the moment he clicked a column header and it sorted exactly as he envisioned, he described it as "magic." ^[extracted] This experience was the pivotal moment that convinced him to pursue programming over mathematics. ^[extracted]

The audience at [[entities/aief2025|AI Engineer World's Fair 2025]] reacted with recognition when Brockman mentioned W3Schools, indicating it remains a widely known and used resource for self-taught developers. ^[extracted]
