# OpenPipe

**OpenPipe** is a company co-founded by [[entities/kyle-corbett|Kyle Corbett]] that does reinforcement learning to make AI agents more reliable. They work primarily with Fortune 500 companies to help them deploy agents in production.

## Focus

OpenPipe's core work is applying RL to improve agent reliability — helping organizations move from prototype agents that occasionally work to production agents that consistently perform correctly. ^[extracted]

## Related Work

Kyle Corbett also maintains the [[concepts/verifiers-toolkit|Verifiers]] open-source toolkit, which provides the infrastructure for training RL agents. OpenPipe's commercial work and the Verifiers toolkit share the same underlying philosophy: making RL for agents accessible and practical. ^[inferred]
