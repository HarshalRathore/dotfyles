# Ido Pesok

Ido Pesok is an engineer at [[entities/vercel|Vercel]] working on [[entities/v0|vZero]], a full-stack vibe coding platform described as "the easiest and fastest way to prototype, build on the web, and express new ideas." ^[extracted]

## AIEF2025 Talk: Evals Are Not Unit Tests

At the AI Engineer World's Fair 2025, Ido delivered a talk on evaluation practices at the application layer, using a basketball court metaphor to illustrate how teams should think about building eval datasets. ^[extracted] Key contributions from the talk include the principle of putting constants in data and variables in tasks, advocating deterministic pass/fail scoring, and integrating evals into CI/CD pipelines. ^[extracted]

## vZero Platform

vZero is Vercel's full-stack vibe coding platform. At the time of the talk, it had crossed 100 million messages sent and had recently launched GitHub Sync — enabling users to push generated code to GitHub, pull changes from GitHub into chat, and switch branches or open PRs to collaborate with teams. ^[extracted]
