# Seth Juarez

[[entities/microsoft|Microsoft]] engineer who presented live demos of [[entities/github-copilot|GitHub Copilot]]'s [[concepts/peer-programming|agentic capabilities]] at the AI Engineer World's Fair 2025 keynote by [[entities/asha-sharma|Asha Sharma]]. ^[extracted]

## Demos Presented

1. **Copilot Spaces** — Grounded Q&A about a multi-agent project using files and instructions. ^[extracted]
2. **Agentic Task Assignment** — Copilot autonomously writing a README from a GitHub Issue and creating a PR. ^[extracted]
3. **Deep Dive with Amelie (M-L-E)** — Copilot delegating to a Machine Learning Engineer agent that builds a predictive model and self-corrects errors. ^[extracted]
4. **Local AI deployment** — Running models and agents locally, not just in the cloud. ^[extracted]
