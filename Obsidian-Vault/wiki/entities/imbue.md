# Imbue

Imbue is an AI agent company focused on building robust, useful AI agents, with current emphasis on software agents for coding workflows.

## Sculptor

Imbue's primary product is [[entities/sculptor|Sculptor]], an experimental coding agent environment designed to bridge the gap between AI-generated prototype code and production-ready code. Sculptor provides synchronous, immediate feedback on generated code rather than waiting until pull request review.^[extracted]

Sculptor is currently in research preview, with features subject to change before public release.^[extracted]

## Philosophy

Imbue's approach to AI coding centers on building user trust through AI-to-AI verification — using one AI system to evaluate the output of another. Their quality model defines high-quality code as code with few or no problems, and focuses on identifying problems as early as possible in the development cycle.^[extracted]
