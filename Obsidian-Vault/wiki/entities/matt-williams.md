# Matt Williams

Author and writer at [The Shape of the System](https://shapeofthesystem.com). ^[extracted]

## Known Work

- **Engineering for Bounded Cognition** (February 2026) — Essay arguing that software engineering is about designing systems that work within the human mind's ~4-chunk working memory limit. ^[extracted]
  - Recommended via tweet by @badlogicgames (Mario Zechner) on June 28, 2026, receiving 594 likes and 32,975 views. ^[extracted]

## Notes

The Shape of the System hosts a series of field notes on building software for the way minds actually work. The bounded cognition essay is described as the conceptual root of this series, with the full set of practical moves available as a manifesto on the same site. ^[extracted]

Limited biographical information available. Primary public presence is at [[references/engineering-for-bounded-cognition|shapeofthesystem.com]].
