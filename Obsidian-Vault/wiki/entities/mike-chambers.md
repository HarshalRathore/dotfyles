# Mike Chambers

Developer Advocate for Amazon Web Services based in Queensland, Australia. Specializes in generative AI, having previously worked in machine learning. ^[extracted]

## Key Contributions

### Fundamentals of LLMs Course

Co-created the "Fundamentals of LLMs" course on DeepLearning.AI alongside Dr. Andrew Ng and colleagues from AWS. The course was the first of its kind and has been taken by over 370,000 people. It includes free access to an AWS Bedrock environment for hands-on practice. ^[extracted]

### AWS Bedrock Agents Advocacy

Chambers is a prominent advocate for Amazon Bedrock Agents as a fully managed path to production-ready agents at cloud scale. He regularly demos the end-to-end flow from local agent to cloud deployment, emphasizing the simplicity of using Bedrock's managed infrastructure, Lambda for tool execution, and the full agent lifecycle with alias-based versioning. ^[extracted]

### Open-Source SDK for Model-First Agents

Referenced an open-source SDK for developing model-first agents at AI Engineer World's Fair 2025, though it was not covered in his talk due to time constraints. ^[extracted]
