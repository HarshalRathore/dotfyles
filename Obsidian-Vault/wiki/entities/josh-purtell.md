# Josh Purtell

Founder of [[entities/synth-labs|Synth Labs]]. His work focuses on developing [[concepts/stateful-environments|stateful environment]] abstractions that enable production-grade AI agents for vertical applications (finance, accounting, health, legal). ^[extracted]

## Key Ideas

- Popularized the [[concepts/stateful-environments|stateful environments]] pattern for containerizing domain logic separately from agent code. ^[extracted]
- Demonstrated how resettable environments make [[concepts/agent-tree-search|agent tree search]] (Language Agent Tree Search) practical in production. ^[extracted]
- Advocates for network boundaries between agents and environments as the production-reliable pattern for async multi-agent work. ^[extracted]
