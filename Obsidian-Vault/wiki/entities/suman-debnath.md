# Suman Debnath

AWS developer advocate who introduced [[entities/strands-agents|Strands Agents]] at AI Engineer World's Fair 2025. ^[extracted]

## Strands Agents

Suman is the creator/advocate for Strands, an open-source SDK designed to make AI agents as simple as possible by eliminating scaffolding. The only two inputs required are a model and tools. ^[extracted]

He demonstrated two key capabilities:
1. Out-of-the-box agent creation with built-in tools (file read/write, text-to-speech)
2. MCP server integration for consuming external tool servers as agent capabilities ^[extracted]
