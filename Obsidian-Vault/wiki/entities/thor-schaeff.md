# Thor Schaeff

Thor Schaeff is a Developer Experience engineer at [[entities/elevenlabs|ElevenLabs]], focused on the conversational AI agents platform. ^[extracted] He also has a colleague named Paul who works with him on developer experience, documentation, examples, and product feedback. ^[extracted]

## AIEF2025 Workshop

Thor delivered a full workshop at the AI Engineer World's Fair 2025 titled "[Full Workshop] Building Conversational AI Agents," where he walked attendees through building multilingual conversational AI agents using ElevenLabs' platform. ^[extracted] He covered ElevenLabs' voice agent pipeline architecture, their ASR model, sound effects capabilities, and multilingual support. ^[extracted]

## AIEF2025 Talk: Milliseconds to Magic

Thor co-presented "Milliseconds to Magic: Real-Time Workflows using the Gemini Live API and Pipecat" at AIEF2025 with [[entities/shristha|Shristha]]. ^[extracted] Approaching from the application layer and agent framework direction, Thor argued that voice is a critical and universal building block for the next generation of generative AI. ^[extracted] Together they introduced the four-layer voice AI stack framework and the "stack maturity" thesis — that capabilities migrate down the stack from application code to frameworks to APIs to models over time. ^[extracted]

Thor demonstrated a voice agent he uses daily for managing priorities — a gray area between a to-do list and brainstorming — which exposed model behavior issues including wrong book titles, turn-taking confusion, and context loss on repeated requests. ^[extracted] This demo illustrated the fundamental shift from traditional programming to model-driven application design, where the model does things the code didn't expect — sometimes badly, sometimes well. ^[extracted]
