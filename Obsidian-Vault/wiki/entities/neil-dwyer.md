# Neil Dwyer

Neil Dwyer is CTO and co-founder of **Gabber**, a startup building real-time AI personas for consumer applications. He previously built the **LiveKit Agents platform** from the first line of code and has a long career in real-time media infrastructure.

## Career

- **Gabber** — CTO and co-founder (with his brother Jack). Building real-time AI personas with voice, memory, video inputs, and tool calling, focused on consumer use cases.
- **LiveKit** — Built the LiveKit Agents platform (born from LiveKit's involvement with GPT Voice). Wrote the first line of code on it.
- **Multiplayer gaming startup** — Co-founded with his brother Jack, targeting AAA-style multiplayer for web games with voice. Used LiveKit. Shut down.
- **Uber** — Worked for a couple of years.
- **Bebo** (acquired by Amazon) — Built game streaming infrastructure (OBS-like), ML pipelines for Fortnite spectator effects (detecting kills/victories to overlay effects).
- Deep experience with **GStreamer, WebRTC, RTMP**.

## AIEF2025 Talk

Neil spoke at AIEF2025 on "Serving Voice AI at $1/hr: Open-source, LoRAs, Latency, Load Balancing," sharing Gabber's experience self-hosting Orpheus TTS for consumer voice AI.
