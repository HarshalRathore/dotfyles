# Tomas Reimers
Tomas Reimers is co-founder of [[entities/graphite|Graphite]], an AI code review company. At AIEF2025, he presented two complementary talks: the inner/outer loop model of software development and the AI Entomology framework for classifying bugs in AI-generated code. ^[extracted]

## Key Contributions

- **Inner/outer loop model** — Articulated the two-loop structure of software development and how AI is reshaping it. ^[extracted]
- **Self-driving code review** — Under his leadership, [[entities/graphite|Graphite]] built Diamond, an AI code reviewer achieving a 52% suggestion acceptance rate, higher than human review comments (45–50%). ^[extracted]
- **AI Entomology** — Coined the term for studying bugs in AI-generated code using AI itself. Analyzed 10,000 code review comments across multiple LLMs to derive a 2D bug taxonomy. ^[extracted]
- **Bug Taxonomy** — Developed a 2D classification framework for code review comments based on LLM capability and human desirability, revealing seven distinct categories of feedback. ^[extracted]
- **Human-LLM Reception Gap** — Discovered through design team interviews that the same review comment is welcomed from a human but rejected from an LLM — a fundamental asymmetry in feedback perception. ^[extracted]
- **AI-native toolchain thesis** — Argued that AI's impact must extend beyond the IDE to the entire developer workflow: testing, review, merge, and deploy. ^[extracted]
