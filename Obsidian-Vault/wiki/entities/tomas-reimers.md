# Tomas Reimers

Tomas Reimers is co-founder of [[entities/graphite|Graphite]], an AI code review company. At AIEF2025, he presented the argument that as AI accelerates the inner loop of software development (writing code), the outer loop (testing, reviewing, merging, deploying) becomes the critical bottleneck — and the entire toolchain must become AI-native, not just the IDE. ^[extracted]

## Key Contributions

- **Inner/outer loop model** — Articulated the two-loop structure of software development and how AI is reshaping it. ^[extracted]
- **Self-driving code review** — Under his leadership, [[entities/graphite|Graphite]] built Diamond, an AI code reviewer achieving a 52% suggestion acceptance rate, higher than human review comments (45–50%). ^[extracted]
- **AI-native toolchain thesis** — Argued that AI's impact must extend beyond the IDE to the entire developer workflow: testing, review, merge, and deploy. ^[extracted]
