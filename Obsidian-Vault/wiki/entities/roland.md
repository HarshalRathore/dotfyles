# Roland

**Roland** is from **ex-xAI** and works in agent infrastructure. He presented multiple concepts at the AI Engineer World's Fair 2026, including agent identity as log, Polygraph, and agent recipes. ^[extracted]

## Presentations at WF2026

Roland presented on: ^[extracted]

1. **Agent identity as log** — the concept that an agent's identity is its append-only event history, decoupling identity from runtime, model, and tools
2. **Polygraph** — a tool that solves space (repo-bound context) and time (amnesia) constraints by capturing agent traces across repos
3. **Agent recipes** — portable, versioned, provider-agnostic git repositories encoding maker taste
