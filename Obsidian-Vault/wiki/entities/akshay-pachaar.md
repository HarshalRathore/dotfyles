# Akshay Pachaar

**Akshay Pachaar** (@akshay_pachaar on X) is a technical writer who has produced comprehensive guides to open-source AI agent frameworks.

## Hermes Agent Masterclass

On May 13, 2026, Pachaar published the **Hermes Agent Masterclass**[^1] — a deep-dive into [[entities/nous-research|Nous Research]]'s Hermes Agent framework. The guide covers:

- Three-tier memory architecture (Markdown files → SQLite FTS → pluggable providers)
- Self-evolving skills with automatic garbage collection via the Curator
- GEPA (Genetic-Pareto Prompt Evolution) for offline skill optimization
- Multi-agent profile setups with Telegram integration and plain-English cron scheduling

This was Pachaar's first major publication in the AI agent education space, as referenced in [[references/hermes-agent-masterclass|the masterclass page]].
