# Laurie Voss

Laurie Voss is VP of Developer Relations at [[entities/llamaindex|LlamaIndex]]. At the AI Engineer World's Fair 2025, he presented on effective agent design patterns in production, covering the five [[concepts/agent-design-patterns|agent design patterns]] codified by [[entities/anthropic|Anthropic]] and how [[entities/llamaindex|LlamaIndex]] implements them via its Workflows abstraction.

## Key Contributions

- Articulated the principle that LLMs excel at turning large bodies of text into smaller ones — a useful design heuristic for choosing agent applications. ^[inferred]
- Presented the five Anthropic agent design patterns (chaining, routing, parallelization, orchestrator workers, evaluator-optimizer) as a practical taxonomy for building production agent systems.
- Demonstrated how LlamaIndex Workflows implement each pattern using event-driven Python with type annotations.
- Argued that the largest LLM market is integration into existing software pipelines, not chatbots.
