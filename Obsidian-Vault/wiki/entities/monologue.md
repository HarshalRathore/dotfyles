# Monologue

**Monologue** is a speech-to-text app built by [[entities/every-company|Every]], showcased at AIEF2025 as an example of what a single engineer can build with AI agents.

## Description

Similar to Whisper Flow (or "Super Whisper"), Monologue provides speech-to-text functionality. Dan Shipper described it as "beautifully done" with "a lot of stuff to it" — it's not a simple app despite being built by one person.

## Engineering

Built by a single engineer with thousands of users. Consistent with Every's model of single-developer products, almost all code written by AI agents.
