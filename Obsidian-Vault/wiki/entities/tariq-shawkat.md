# Tariq Shawkat

**Tariq Shawkat** is the CEO of **Sonar** and presented the **ACDC (Guide-Verify-Solve)** framework for AI agent reliability at the AI Engineer World's Fair 2026. ^[extracted]

## ACDC Framework

Tariq presented the ACDC framework, arguing that code generation alone is insufficient and that agents must be surrounded by three compounding loops: ^[extracted]

1. **Guide** — structured prompts, context, and constraints
2. **Verify** — multi-layered verification (agentic, CI, code maintenance)
3. **Solve** — remediation of issues surfaced by verification

Multi-layered verification reduces production outages by **44%**. Guide-verify-solve reduces issues by **92%**. ^[extracted]

## Sonar

Sonar launched a product around guided agent workflows at the time of this talk. The product focuses on providing shared context between agents and humans to reduce the explanation burden. ^[extracted]
