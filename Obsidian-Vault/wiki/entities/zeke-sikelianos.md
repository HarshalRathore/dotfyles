## Zeke Sikelianos

Zeke Sikelianos works at Replicate and presented "Design like Karpathy is watching" at AIEF2025. He is known on GitHub and X as "Zeke."

### AIEF2025 Talk

His talk centered on Andre Karpathy's influence on how AI-native products should be designed, using Karpathy's Menugen project as a case study for developer experience challenges. He covered Replicate's response to these challenges: LLM-friendly documentation (llms.txt), curl-first API design, OpenAPI schemas, and MCP server integration.
