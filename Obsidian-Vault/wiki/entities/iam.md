# IAM

## Basic Information
- Type: product
- Source: introduction_804367

## Description
AWS Identity and Access Management (IAM) is a core service of [[entities/aws]] that provides fine-grained access control and permissions management for all AWS resources. It allows administrators to create and manage users, groups, and roles to securely grant or restrict access to specific AWS services and resources. IAM is a foundational security component, recommended as a best practice for securing an AWS environment before provisioning infrastructure. Its use is essential for creating dedicated administrative users with specific access types for operational tasks, separating duties from the high-privilege root account.
