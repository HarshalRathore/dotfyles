# Weaveworks’ Sock Shop

## Basic Information
- Type: project
- Source: microservices-demo-projects_ac9b1f

## Description
Weaveworks’ Sock Shop is an open-source, microservices-based online store application designed as a demonstration platform. The application is implemented as eight distinct microservices: front-end, orders, payment, user, catalog, cart, shipping, and queue components. These services are developed using Spring Boot (Java), Go kit, and Node.js, showcasing polyglot microservice architecture. The project provides complete deployment assets, including Dockerfiles for each service and Kubernetes YAML or Helm deployment scripts. It was originally created to demonstrate Weave Net and Kubernetes capabilities in hands-on workshops. The application runs as Docker containers by default and is explicitly used for practicing microservices deployment and Kubernetes networking.
