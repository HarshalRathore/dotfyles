# Jeff Atwood

Jeff Atwood is a software engineer, writer, and entrepreneur, best known as the co-founder of [[entities/stack-overflow|Stack Overflow]] (with [[entities/joel-spolsky|Joel Spolsky]]). He authored the influential "Coding Horror" blog, which shaped a generation of software developers' thinking about programming practices, code quality, and software craftsmanship. ^[inferred]

## Notable Quote

> "The best code is no code at all."

Quoted by [[entities/chris-kelly|Chris Kelly]] in his AIEF2025 talk to argue that AI-generated code is not a net positive by volume — every line of generated code carries maintenance burden, making code minimization more valuable, not less, in the AI era. ^[extracted]
