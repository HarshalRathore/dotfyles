Cron is an engineer at Luma AI who manages the company's public API. He built out Luma's SDK and developer relations program, enabling external developers to integrate Dream Machine's image and video generation capabilities into their own applications.

## API Developer Experience

Keegan McCallum gave Cron a public shout-out during his AIEF2025 presentation, noting that developers "don't need to do any kind of crazy prompt engineering" because Cron's team has "taken care of that for you." The API accepts raw user prompts and generative media inputs, returning images and videos that meet users' needs.

Cron's work on the SDK and developer relations has been critical to Luma's API strategy — making it easy for external developers to integrate multimodal generation without complex prompt engineering or infrastructure setup.
