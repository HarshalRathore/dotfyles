# Elvis Sun

Elvis Sun (@elvissun) is an independent builder describing himself as "2x dad building the agentic PR stack in public." He has 46K followers on Twitter and is verified.

## Products

- **newsjack.sh** — "The operating system for agentic PR" — an open-source CLI tool for agentic PR workflows. Provides an install script and compiled binary for macOS and Linux. ^[extracted]
- **medialyst.ai** — AI-powered media list builder for PR professionals, founders, and communications teams. Matches journalists to stories using AI agents. ^[extracted]
- **LFD (Loss Function Development)** — A methodology and skill (`lfd-design`) for designing loss functions for autonomous agent loops. Published as a comprehensive playbook on Twitter in June 2026. ^[extracted]

## Key Contributions

Elvis published a widely-read (577K views) playbook on Loss Function Development — arguing that the bottleneck in autonomous agent development is not writing specs but designing loss functions with proper targets, constraints, instruments, and forced entropy. ^[extracted]
