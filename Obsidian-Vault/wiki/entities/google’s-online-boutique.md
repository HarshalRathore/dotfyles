# Google’s Online Boutique

## Basic Information
- Type: project
- Source: microservices-demo-projects_ac9b1f

## Description
Google’s Online Boutique, also known as Hipster Shop, is an open-source e-commerce demonstration application composed of 10 distinct microservices. Each service is implemented in a different programming language—including Go, Node.js, Java, C#, and Python—and they communicate using the gRPC framework. The project provides containerized builds via Dockerfiles and deployment manifests for Kubernetes, such as Helm charts, making it ready for deployment on clusters like GKE. It serves as a popular reference architecture and educational tool for learning multi-tier application deployment on Kubernetes.
