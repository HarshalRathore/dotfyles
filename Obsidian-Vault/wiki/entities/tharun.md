# Tharun

**Tharun** is a research engineer at [[entities/datalab|Datalab]]. He is a key example of the generalist engineer model that [[entities/vik-paruchuri|Vik Paruchuri]] advocates for. ^[extracted]

## Surya OCR3

Tharun and Vik Paruchuri co-developed [[entities/surya|Surya OCR3]] end-to-end. The scope included: ^[extracted]

- Talking to customers to understand requirements
- Reading papers and designing architecture
- Prototyping and building a data pipeline library
- Model training (noting "it's always 90% data cleaning")
- Writing inference code
- Connecting to repos and integrating into products for all customers

This 2-person team accomplished work that would require 4-10 teams in a large company, with no context loss from handoffs. ^[inferred]
