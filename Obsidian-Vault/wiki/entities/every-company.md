# Every

**Every** is an AI-native company founded and run by [[entities/dan-shipper|Dan Shipper]]. It serves as a living example of what's possible when a small team uses AI agents for nearly all engineering work.

## Company Profile

| Metric | Value |
|--------|-------|
| Team size | 15 people |
| Business units | 6 |
| Software products | 4 |
| Paying subscribers | 7,000+ |
| Free subscribers | 100,000+ |
| MRR growth | Double digits monthly (6+ months) |
| Total funding | ~$1M |
| Code written by AI | 99% |

## Products

### Quora
An AI email management app that summarizes incoming emails and serves as an email assistant — you can ask questions like "when's my AI engineer talk today?" and get answers directly. Built primarily by one engineer.

### Monologue
A speech-to-text app similar to Whisper Flow. Beautifully designed, complicated functionality, built by one engineer with thousands of users.

### Spiral
A larger production application built by a single developer.

## Engineering Model

Every operates on principles that make it a "lab for what's possible" in AI-native engineering:

- **99% AI-written code**: No one handwrites code. AI agents (Claude Code, Codex, Droid) write virtually everything.
- **Single-developer apps**: Each product is built and maintained by one developer, not a team.
- **Compounding engineering**: The four-step loop (plan, delegate, assess, codify) makes each feature easier to build than the last.
- **Parallel agent work**: Engineers productively use multiple agent panes simultaneously.

## Significance

Every demonstrates that small, capital-efficient teams can build and run multiple production-grade software products when AI agents handle the vast majority of coding. Shipper positions it as a "little bit of a lab for what's possible" — not a finished playbook, but a working example of AI-native company building in practice.
