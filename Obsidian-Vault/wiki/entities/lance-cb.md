# LanceCB

LanceCB is a company building data tools for machine learning and data science, founded by [[entities/calvin-qi|Calvin Qi]]. The company focuses on data that doesn't fit neatly into [[entities/pandas|Pandas]] data frames. ^[extracted]

## Background

Calvin Qi has been building data tools for ML and data science for about 20 years and is a co-author of the Pandas library. LanceCB extends this work to the data shapes that Pandas doesn't handle well. ^[extracted]
