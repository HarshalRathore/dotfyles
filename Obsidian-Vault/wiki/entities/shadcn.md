# shadcn

**shadcn** (shadcn/ui) is a popular UI component library built on Radix UI primitives and styled with Tailwind CSS.

## Role in Modern AI App Stacks

shadcn provides accessible, customizable UI components that integrate seamlessly with Tailwind CSS. It is a standard component of the modern TypeScript full-stack, used alongside Tailwind for styling in Next.js applications. ^[extracted]
