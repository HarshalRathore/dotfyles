# Geoffrey Litt

Geoffrey Litt (@geoffreylitt) is an AI engineer and open-source contributor known for practical tools that improve human-agent collaboration in software development.

## Contributions

- **[/explain-diff](https://gist.github.com/geoffreylitt/a29df1b5f9865506e8952488eac3d524)** — An agent skill that generates structured, human-readable code explanations in HTML or Notion format, using principles of background context, intuition-first delivery, literate diffs, and comprehension quizzes.
- Published widely-shared threads on understanding agent-written code, cognitive debt, and the pedagogy of AI-assisted programming.

## Key Ideas

- Understanding code written by agents is essential for creative participation, not just verification ^[extracted]
- Cognitive debt (co-developed with Margaret Storey and @simonw) compounds when developers don't keep up with understanding their own systems ^[extracted]
- Educational techniques — background-first explanations, interactive micro-worlds, spaced repetition quizzes — are the most effective tools for building human fluency alongside AI ^[extracted]
