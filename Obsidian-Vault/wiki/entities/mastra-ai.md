# Mastra.ai

An AI workflow framework co-founded by [[entities/sam-bhagwat|Sam Bhagwat]] (previously co-founder of [[entities/gatsby|Gatsby]]). Mastra provides a fluent, grokkable API for defining agent workflows as readable sequential code rather than graph-based node-and-edge configurations. ^[extracted]

## Design Philosophy

- **Fluent syntax over graph theory** — workflows are defined as readable sequential code (do A then B then C) rather than requiring developers to learn graph theory concepts (nodes, edges, terminals) ^[extracted]
- **Composition over choice** — frameworks should support composing agents as workflow steps, workflows as agent tools, agent supervisor models, and nested workflows ^[extracted]
