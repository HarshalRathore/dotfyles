# Mozilla AI

Mozilla's AI division, operating in the open source space. Provides open source AI tooling and supports the open source AI stack. Its stated goal is to enable the open source community to be "at the same table as a Sam Altman when talking about AI moving forward." ^[extracted]

## Leadership

- **CEO:** [[entities/john-dickerson|John Dickerson]], former co-founder and chief scientist of [[entities/arthur-ai|Arthur AI]] ^[extracted]

## Products

- **AnyAgent** — An open source, unified interface for multi-agent systems. Allows users to try different multi-agent system frameworks under a single API, rather than being locked into one framework. ^[extracted] Not monetized. ^[extracted]
