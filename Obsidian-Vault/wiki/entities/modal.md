# Modal

A cloud platform providing serverless GPU compute for AI and ML workloads, including training, fine-tuning, inference, and batch processing. Modal offers sandboxed execution environments that are well-suited for running untrusted code or multi-tenant AI workloads. ^[inferred]

## LLM Almanac

Modal's [[entities/charles-frye|Charles Frye]] built and maintains the LLM Almanac at modal.com/llm-almanac — a comprehensive open benchmark comparing [[entities/vllm|vLLM]], [[entities/sglang|SGLang]], and TensorRT-LLM across ~10 models and ~10 context lengths on H100 GPUs. ^[extracted]

The project started from Modal's consulting work answering client questions about inference performance. What initially took days was automated to ~15 minutes per query, then pre-computed into the public Almanac. ^[extracted]

Key findings from the Almanac:
- Prefill-dominated workloads achieve ~4x higher throughput than decode-dominated workloads on H100s. ^[extracted]
- First token under 300ms (Doherty threshold) is achievable on Qwen3 8B and Gemma3 27B. ^[extracted]
- vLLM shows stronger optimization for Llama and Qwen model families; Qwen3 had initial compatibility issues with SGLang's 8-bit quant. ^[extracted]

## Usage in Hackathon

The Tab RL team used Modal to host sandboxed reinforcement learning training environments. After reward functions were generated by foundation models, training jobs were dispatched to Modal's infrastructure for execution, with results (trained policies) returned to the browser for visualization. ^[extracted]
