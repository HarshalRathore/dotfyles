# Quora (Every)

**Quora** is an AI email management app built by [[entities/every-company|Every]], a 15-person AI-native company run by [[entities/dan-shipper|Dan Shipper]]. It was showcased at AIEF2025 as an example of what a single engineer can build with AI agents.

## Features

- **Email summarization** — On the left side, summarizes all incoming emails so users can read their inbox efficiently
- **Email assistant** — On the right side, an AI assistant you can ask questions about your emails. Example: "when's my AI engineer talk today?" — the assistant answers directly from email content

## Engineering

Built primarily by a single engineer with one or two contractors helping in specific areas. Almost all code is written by AI agents, consistent with Every's 99% AI-written code model.

## Significance

Quora demonstrates that a complex, production-grade email application with AI features can be built and maintained by one person when AI agents handle the vast majority of coding work.
