# Bespoke Labs

**Bespoke Labs** is a startup founded by [[entities/ryan-martin|Ryan Martin]] focused on building the best open-source reasoning datasets.

## OpenThoughts Project

Ryan Martin created the OpenThoughts project at Bespoke Labs, which produced OpenThoughts 3 — the state-of-the-art open reasoning dataset recipe. The project involved 1,000 experiments, 5,000+ datasets, and 3,000 models. ^[extracted]

## Open-Source Tools

- **Curator** — Library for synthetic data generation
- **Evalchemy** — Evaluation library with sharding and parallelism support
