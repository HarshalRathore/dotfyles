# Red Hat

Red Hat is an enterprise open-source software company (an IBM subsidiary) with an established AI Business Unit. Its AI work spans inference infrastructure, evaluation tooling, and model deployment on RHEL. ^[extracted]

## AI Evaluation Contributions

- **[[entities/promptfu|PromptFu]]** — open-source safety evaluation tool for LLMs
- **[[entities/vllm|VLLM]] integration** — inference runtime deployment on RHEL for model serving and benchmarking
- **[[entities/guidellm|GuideLLM]]** — system performance benchmarking tool associated with the VLLM ecosystem
- **Enterprise infrastructure** — provides RHEL-based systems with GPU support (L4) for AI workloads ^[extracted]
