# Sizzy

A browser specifically designed for front-end developers, created by [[entities/kitze|Kitze]]. Not intended to replace a general-purpose browser for everyday browsing, but rather a specialized development tool analogous to Photoshop — built to assist with front-end development workflows. ^[extracted]

## Purpose

Sizzy addresses the needs of developers who need to test and preview front-end changes in real time, providing capabilities beyond what standard browsers offer for development workflows. Kitze describes it as "a tool like Photoshop that's like helping you in a lot of ways to do front-end development." ^[extracted]
