# Peter Steinberger

Peter Steinberger (@steipete) is an independent iOS/macOS developer and open-source author. He created [[entities/summarize]] (CLI + Chrome Side Panel for LLM-powered web/file/media summarization), [[entities/birdclaw]] (local-first Twitter/X archive and operator console), and the `bird` CLI for browser-cookie-backed Twitter/X access.

## Known for

- **[[entities/summarize]]** — CLI + Chrome Side Panel for clean extraction and LLM summarization of URLs, files, YouTube, podcasts. MIT licensed, 6.3k stars.
- **[[entities/birdclaw]]** — Local-first Twitter memory in SQLite: archives, DMs, likes, bookmarks, with a CLI + web UI
- **`bird` CLI** — Browser-cookie-backed Twitter/X reads and writes, used as a transport by birdclaw
- **sweetistics** — Earlier Twitter archive and sync tooling that birdclaw's architecture draws from
- **PSPDFKit** — Former co-founder and iOS SDK developer (before it was acquired)

## Open-source work

- [summarize](https://github.com/steipete/summarize) — 6.3k stars, MIT license, actively maintained
- [birdclaw](https://github.com/steipete/birdclaw) — 1.5k stars, MIT license, actively maintained
- [bird](https://github.com/steipete/bird) — Browser-cookie-backed X/Twitter CLI
