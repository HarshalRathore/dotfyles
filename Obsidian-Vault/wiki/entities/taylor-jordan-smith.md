# Taylor Jordan Smith

AI developer advocate (with roles spanning evangelist, technical marketing, and related titles) at [[entities/red-hat|Red Hat]]'s AI Business Unit. Presented the "Strategies for LLM Evals (GuideLLM, lm-eval-harness, OpenAI Evals Workshop)" workshop at AI Engineer World's Fair 2025. ^[extracted]

## Key Contributions

- Introduced the [[concepts/llm-evaluation-pyramid|LLM Evaluation Pyramid]] — a tiered framework for evaluating LLMs in production across system performance, formatting, factual accuracy, safety/bias, and custom evals ^[extracted]
- Led a hands-on workshop using [[entities/guidellm|GuideLLM]] for system performance benchmarking, [[entities/lm-eval-harness|lm-eval-harness]] for factual accuracy, and [[entities/promptfu|PromptFu]] for safety/bias evaluation ^[extracted]
- Deployed an IBM Granite model using [[entities/vllm|VLLM]] on RHEL infrastructure for the workshop exercises
