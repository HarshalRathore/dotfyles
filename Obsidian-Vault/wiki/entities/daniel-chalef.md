# Daniel Chalef

Daniel Chalef is a representative of [[entities/zep|Zep]] who presented at the AI Engineer World's Fair 2025.

## AIEF2025 Talk: "Stop Using RAG as Memory"

Chalef delivered a talk arguing that agent memory should be modeled after the business domain rather than generated as arbitrary facts stored in vector databases. He demonstrated Zep's domain-aware memory approach using developer-defined entity types and a temporal graph called [[concepts/graffiti-temporal-graph|Graffiti]]. ^[extracted]
