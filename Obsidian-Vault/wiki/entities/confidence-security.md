# Confidence Security

Confidence Security (also referred to as CONFSEC) is a company founded by [[entities/jmo|Jmo]] that is building privacy-preserving remote compute infrastructure — the same capabilities as [[concepts/private-cloud-compute|Apple's Private Cloud Compute]] but for organizations not on Apple's platform. ^[extracted]

The company builds on the same technical building blocks available in the broader ecosystem: [[concepts/oblivious-http|Oblivious HTTP]], [[concepts/blind-signatures|blind signatures]], [[concepts/remote-attestation|remote attestation]] via [[concepts/confidential-gpu|TPMs/vTPMs]], transparency logs (Sigsum/Sigstore), and confidential VMs with confidential GPUs. ^[extracted]
