## Core Content
This source provides a curated list of open-source microservices demo projects chosen for practical Kubernetes learning. It evaluates projects based on moderate complexity, realistic architecture, and included deployment files (Dockerfiles and Kubernetes YAML/Helm charts). The goal is to prepare users for deploying multi-service applications on AWS EKS with Terraform. The projects selected—[[entities/google’s-online-boutique]], [[entities/weaveworks’-sock-shop]], [[entities/cisco’s-martianbank]], and [[entities/microsoft’s-eshop-reference-application]]—span e-commerce and fintech domains. They offer hands-on experience with core cloud-native concepts like [[concepts/microservices-architecture]], [[concepts/containerization]], [[concepts/service-discovery]], and monitoring, using tools such as [[entities/helm]] and [[entities/docker-compose]].
