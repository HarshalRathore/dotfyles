# AIEF2025 - Using OSS models to build AI apps with millions of users — Hassan El Mghari

**Speaker:** Hassan El Mghari, Developer Relations lead at Together AI
**Event:** AI Engineer World's Fair 2025
**Video:** https://www.youtube.com/watch?v=gcseUQJ6Gbg

## Overview

Hassan El Mghari shares his process for building open-source AI applications from idea to production, including demos of apps that have reached millions of users. He covers his tech stack, architecture patterns, ideation methods, and practical advice for builders.

## Key Topics

1. **Speaker introduction** — Hassan's background building ~1 AI app per month for 4 years, all open source
2. **Together AI overview** — Open source model inference platform
3. **Why now to build** — Lowered barriers from AI tools + improving models
4. **App demos** — Multiple apps with user counts (40K to 1M+)
5. **Architecture patterns** — Simple 4-step architecture (input → AI call → store → show)
6. **Tech stack** — Together AI, Next.js, TypeScript, Neon, Clerk, Prisma, shadcn/Tailwind, S3, Plausible, Helicone, Vercel
7. **Ideation process** — Running idea lists, source-driven discovery, model-driven interruption
8. **Naming and marketing** — Strategic naming and distribution

## Notable Apps Shown

- Glasses picker (Amazon API + AI)
- Git commit message writer (40K installs)
- Text-to-app builder (1M+ users, 5M+ requests)
- Image generation app (1M+ users, 48M+ images)
- Napkins (hand-drawn → web app, 40K users)
- Resume-to-site builder
- Menu visualizer
- AI tutor app
- OCR tools

## Key Quote

> "It's a numbers game — you kind of have to try out a lot of different stuff."
