# AIEF2025 - Mastering Engineering Flow with Windsurf - Eashan Sinha, Windsurf

**Speaker:** Eashan Sinha, Engineer at Windsurf
**Video:** https://www.youtube.com/watch?v=W_5tzQY-hVs
**Event:** AI Engineer World's Fair 2025

## Summary

Eashan Sinha presented on how developers can make the most of their development experience with Windsurf's AI Flows paradigm, and how AI agent flows make developers better. The talk covered the evolution from co-pilots to agents, Windsurf's approach to flow awareness, the Cascade agent's context engine, and workflows as a bridge between agent unpredictability and deterministic execution.

## Key Topics

- **Agent-Developer Relationship** — Treating agents as teammates, not tools; the Tom and Jerry vs. iconic duo metaphor
- **Evolution of AI Coding Tools** — DIY (2022) → Co-pilots (2022-23) → Agents (2024+) → AI Flows (2024+)
- **Flow Awareness** — Tracking user edits, terminal commands, clipboard, and file activity to predict next steps
- **Context Engine** — Multi-tool context retrieval beyond RAG/embedding search for better code understanding
- **AI Flows** — Unified interface combining chatbot collaboration with agent autonomy
- **Workflow Integration** — Bridging agent unpredictability with workflow determinism
