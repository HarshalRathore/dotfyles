# Code Is Cheap

**Code Is Cheap** is the economic principle that AI agents have dramatically reduced the marginal cost of writing code, making it feasible to prototype risky ideas freely and run far more experiments than was previously possible.

## The Economic Shift

When code is expensive — when it takes days or weeks to write, review, and deploy — organizations must be conservative about what they build. The starting energy to try something is high, so only the most confident ideas get pursued.

When code is cheap — when an AI agent can produce a working prototype in hours — the calculus changes:

- You can prototype risky ideas without significant investment
- You can run more experiments in the same timeframe
- The starting energy to try something is dramatically lower
- You can say "go do some research on this big refactor I might want to do" and then go work on something else while the agent investigates

## Practical Implications

### More Experiments, More Progress

Shipper identifies this as "a really big deal": the ability to run more experiments directly translates to more progress. When you can cheaply explore multiple directions, you find the right one faster.

### Risk-Tolerant Innovation

When code is cheap, teams can afford to be wrong more often. Failed experiments cost little, so teams explore more aggressively. This leads to discovering opportunities that conservative, high-cost engineering would never have attempted.

### Parallel Exploration

Combined with [[concepts/parallel-agents|parallel agent work]], the cheap-code principle means a single developer can explore multiple ideas simultaneously — building a demo for one idea while another agent researches a refactor for a different project.

## Relationship to Other Concepts

Code being cheap is the economic foundation that makes [[concepts/demo-culture|demo culture]] possible. It's also what enables [[concepts/compounding-engineering|compounding engineering]] to work at scale — if each iteration of the plan-delegate-assess-codify loop were expensive, the compounding effect would be too slow to matter.
