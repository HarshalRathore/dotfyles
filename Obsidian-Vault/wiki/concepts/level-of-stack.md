# Level of Stack

**Level of Stack** is the concept that AI agents are pushing programming up a level of abstraction. Engineers are moving from Python, JavaScript, and other scripting languages up into English as the primary programming language.

## The Abstraction Progression

This is the next step in the [[concepts/evolution-of-programming|evolution of programming]]:

1. **Hardware** — Switchboards, physical wiring
2. **Punch cards** — Physical machine code
3. **Assembly** — Mnemonic abbreviations of machine instructions
4. **High-level languages** — Python, JavaScript, Rust, etc.
5. **Natural language** — English instructions to AI agents

Each step raises the level of abstraction, moving closer to expressing intent directly rather than expressing intent through an intermediate formalism.

## What Changes at a Higher Level

When programming moves from code to English:

- **The skill shifts** from syntax mastery to problem decomposition and clear communication
- **The output is still code** — the agent translates English into executable programs
- **The feedback loop changes** — instead of reading compiler errors, you read working software
- **Tacit knowledge becomes more shareable** — prompt patterns are easier to share than code patterns

## Implications for Engineering Organizations

At [[entities/every-company|Every]], this shift is already in practice. With 99% of code written by AI agents, the engineers at Every are primarily communicating in English — describing what they want, not how to build it in a specific language. This is not "vibe coding" (rapid prototyping without planning); it's a fundamental reorganization of the engineering workflow around a higher level of abstraction.

## Relationship to Other Concepts

The level-of-stack shift enables [[concepts/tacit-knowledge-codification|tacit knowledge codification]] because prompt patterns (English instructions) are inherently more portable and shareable than code patterns (language-specific implementations). It's also the defining characteristic of an [[concepts/ai-native-company|AI-native company]] — a company built around the premise that English is the primary programming language.
