# AWS Free Tier

## Definition
The AWS Free Tier is a program offered by Amazon Web Services (AWS) that provides new customers with a limited amount of usage for specific AWS services at no cost for the first 12 months following account creation.

## Key Characteristics
- Provides 12 months of free usage for eligible services.
- Offers a combination of always-free, 12-month free, and trial-tier resources.
- Intended to lower the barrier to entry for new users to learn, experiment, and build.
- Requires a valid credit card for account verification and to handle any usage beyond the free tier limits.


## Applications
- Learning and experimenting with AWS services without incurring initial costs.
- Hosting development and testing environments, such as a demo Kubernetes cluster.
- Building and deploying small-scale applications or prototypes for evaluation.
- Completing tutorials and educational projects that utilize cloud infrastructure.
