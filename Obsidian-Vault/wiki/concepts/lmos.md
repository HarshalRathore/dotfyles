# LMOS

**LMOS** (Language Model Operating System) is one of the earliest proposed standard models for AI engineering — a framework for thinking about how language models function as the core operating system of modern AI applications.

## Origin

LMOS was proposed by [[entities/karpathy|Andrej Karpathy]] in 2023 and updated by [[entities/swyx|swyx]] for 2025 to reflect the evolution of the AI engineering landscape.

## 2025 Updates

swyx's 2025 update to LMOS includes:

1. **Multimodality** — Language models are no longer text-only; they handle images, audio, video, and code
2. **MCP as default protocol** — [[concepts/model-context-protocol|MCP]] has become the standard protocol for connecting models with the outside world
3. **Tool ecosystem** — The expanded set of tools and integrations available to language models

## Position as a Standard Model

LMOS is one of several candidate **standard models** for AI engineering, alongside:
- [[concepts/lnsdlc|LNSDLC]] — AI-adapted Software Development Lifecycle
- [[concepts/spade-pattern|SPADE]] — Sync, Plan, Analyze, Deliver, Evaluate pipeline
- [[concepts/agent-design-patterns|Effective Agent Patterns]] — Received wisdom for building agents
