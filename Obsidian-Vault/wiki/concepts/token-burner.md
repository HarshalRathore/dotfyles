# Token Burner

An employee who consumes the most AI coding tokens in an organization, tracked on "vibe coding leaderboards" as a proxy for adoption and productivity. ^[extracted]

## The Concept

Companies like [[entities/shopify|Shopify]] implement vibe coding leaderboards that count token consumption by employees. Counterintuitively, employees burning the most tokens are considered more valuable — not because they're wasteful, but because they're actively adopting the new skill of AI-assisted development. ^[extracted]

## Interpretation

High token consumption correlates with:

- Active use of AI coding tools
- Willingness to iterate and experiment
- Engagement with the vibe engineering workflow
- Productivity through AI acceleration

This inverts the traditional cost-center view of AI token usage. Instead of minimizing token spend, organizations that embrace AI coding treat token burn as a leading indicator of adoption and output velocity. ^[inferred]

## Caveats

Token burn alone is not a quality metric. A vibe engineer who burns tokens thoughtfully (iterating, correcting, refining) is more valuable than one who burns tokens randomly. ^[inferred] The leaderboard measures adoption, not correctness.
