# Open Source AI Apps

Building AI applications as **open source projects** provides distribution, community contribution, and transparency advantages.

## Distribution Through Open Source

Open source AI apps benefit from:

- **GitHub stars** — Organic visibility and credibility signal
- **Community contributions** — Others can improve and extend the code
- **Self-service exploration** — Users can dig into the code themselves
- **Network effects** — Open source projects are more likely to be shared and recommended

## Track Record

Several open-source AI apps have achieved significant scale:

- Git commit message writer: 40,000+ installs, community contributors
- Text-to-app builder: 1M+ users, 5M+ requests, 1M+ apps built
- Image generation app: 1M+ users, 48M+ images generated

## Philosophy

Hassan El Mghari builds everything as open source, encouraging users to explore the code. This reflects a broader principle: open source AI apps lower the barrier for others to learn from, extend, and build upon them. ^[extracted]

## Relationship to Simplicity

Simple architectures (single API call, minimal infrastructure) are particularly well-suited to open source because they are easier for others to understand, contribute to, and fork. ^[inferred]
