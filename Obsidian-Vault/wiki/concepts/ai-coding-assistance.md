# AI Coding Assistance

**AI coding assistance** refers to AI-powered tools that help developers write code faster — enabling rapid feature development, prototyping, and experimentation. In the context of product development, it serves as an enabler for broad feature exploration by making features cheap to build and cheap to cut. ^[extracted]

## Role in Feature Exploration

The **Incubator for AI** used AI coding assistance to build many different features for their Minute transcription product, testing each with different user groups. The key advantage: because AI coding makes features cheap to build, the team had no sentimental attachment to any particular feature, making it easy to cut features that didn't work. ^[extracted]

This creates a virtuous cycle:

1. AI coding → cheap features
2. Cheap features → broad exploration
3. Broad exploration → right design
4. No sunk cost → easy feature cutting

## Andrew Ng's Broader Point

Andrew Ng noted that writing software, especially prototypes, is becoming cheaper due to AI coding agents and assistants, and that AI features in products make the previously impossible possible. This increases demand for people who can decide what to build — the AI product manager. ^[inferred]
