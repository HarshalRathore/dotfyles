# Lighthouse Scores

Lighthouse scores are a set of performance, accessibility, best practices, and SEO metrics for web pages, generated by Google's Lighthouse auditing tool. AI agents can automatically improve Lighthouse scores by identifying and fixing the underlying issues. ^[extracted]

## Lighthouse Categories

Lighthouse evaluates five categories:
- **Performance**: Core Web Vitals (LCP, FID, CLS), load time, render blocking
- **Accessibility**: WCAG compliance, ARIA attributes, color contrast
- **Best Practices**: HTTPS usage, console errors, CSP headers
- **SEO**: Meta tags, crawlability, structured data
- **PWA**: Service workers, manifest, offline support ^[inferred]

## AI Agent Improvement

In his AIEF2025 demo, Rustin Banks had Jules improve Lighthouse scores as part of a parallel agent workflow. The agent:
1. Ran Lighthouse to identify issues
2. Generated fixes for performance bottlenecks
3. Applied accessibility improvements
4. Verified score improvements ^[extracted]

This demonstrates that Lighthouse improvement is a viable "laundry" task for AI agents — well-defined, measurable, and automatable.
