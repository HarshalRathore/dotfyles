# Voice Dictation in AI Coding Assistants

GitHub Copilot includes voice dictation (Command-I by default) that uses a local model for privacy and speed. It enables hands-free interaction with the AI coding assistant, allowing users to describe changes without touching the keyboard. ^[extracted]

## Features

- **Local model** — Runs locally for privacy; no data sent to cloud models
- **Speed** — Fast response time suitable for interactive use
- **Accuracy** — Good enough for coding-related commands
- **Read-back** — Option to have the model read back the recognized text, useful for accessibility
- **Hey Copilot** — Wake-word activation (mentioned as previously available, usage uncertain)
- **Keyboard shortcut** — Command-I by default, customizable to hold-while-talking then release

## Role in Vibe Coding

Voice dictation is presented as a way to "put that coffee down and just keep vibing" — the ultimate hands-free coding experience. It fits the YOLO vibe coding ethos of staying in flow without interruption. ^[extracted]
